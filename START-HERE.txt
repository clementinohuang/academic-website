本地预览
--------
1. 确保装了 Hugo（Mac: brew install hugo）。
2. 终端 cd 进 my-site 文件夹。
3. 运行:  hugo server
4. 浏览器打开 http://localhost:1313

· 颜色/样式：assets/css/extended/custom.css（所有紫色在顶部 :root）
· 首页头像：static/photo.jpg
